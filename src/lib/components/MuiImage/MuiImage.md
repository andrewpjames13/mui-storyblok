MuiImage example:

```js

const imageStyle = ['height: 150px'];

<MuiImage
  image='https://images.unsplash.com/photo-1477346611705-65d1883cee1e?ixlib=rb-1.2.1&ixid=eyJhcHBfaWQiOjEyMDd9&auto=format&fit=crop&w=3150&q=80'
  imageStyle={imageStyle}
/>
```

MuiImage With Href example:

```js

const imageStyle = ['height: 100px'];
const hrefLink = 'https://woodriverenergy.com/';

<MuiImage
  image='https://lh5.googleusercontent.com/-r16cBsJCTXc/AAAAAAAAAAI/AAAAAAAAAAA/Gt12G-Kt2rQ/s88-p-k-no-ns-nd/photo.jpg'
  imageStyle={imageStyle}
  hrefLink={hrefLink}
/>
```
